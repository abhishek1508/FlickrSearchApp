# FlickrSearchApp
App used to fetch images from Flickr using Compose and KTOR

Dependencies to build this app:

- Kotlin
- Jetpack Compose
- KTOR to make n/w requests
- ViewModel
- Coroutines
- Koin for DI
- Coil to fetch images

For extra credits:

- Landscape mode
- Unit test for `DefaultFlickrApiServiceImpl`

Architectire:

- MVVM
